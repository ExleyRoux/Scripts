1. Install Python3:
	Download and install the latest version of python here: 
		https://www.python.org/downloads/

2. Install Python dotenv:
	Run the following command:
		pip install python-dotenv
		
3. Inseall Python charade:
	Run the following command:
		pip install charade
		
3. To run script:
	3.1. Open a cmd terminal and navigate to the script folder
	3.1. Run the command 'Python3 %script_name%.py' where %script_name% is the filename of the script to be run
